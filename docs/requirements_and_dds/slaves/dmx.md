# DMX SLAVE

**REQ-DMX-100:** The DMX slave shall output a full 512-channel DMX universe continuously.

**REQ-DMX-110:** The DMX slave shall store multiple named universes in NVS — each with a name string and 512 channel values.

**REQ-DMX-120:** The DMX slave shall activate a stored universe on command from master.

**REQ-DMX-130:** The DMX slave shall accept raw channel values directly without requiring a stored universe.

**REQ-DMX-140:** The DMX slave shall continue sending the last known universe if contact with master is lost.

**REQ-ERR-110:** DMX slave shall continue sending last known universe if master contact is lost.
