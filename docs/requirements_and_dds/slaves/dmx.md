# DMX NODE

**REQ-DMX-100:** The DMX node shall output a full 512-channel DMX universe continuously.

**REQ-DMX-110:** The DMX node shall store multiple named universes in NVS — each with a name string and 512 channel values.

**REQ-DMX-120:** The DMX node shall activate a stored universe on command from master.

**REQ-DMX-130:** The DMX node shall accept raw channel values directly without requiring a stored universe.

**REQ-DMX-140:** The DMX node shall continue sending the last known universe if contact with master is lost.

**REQ-ERR-110:** DMX node shall continue sending last known universe if master contact is lost.
